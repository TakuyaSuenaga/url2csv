# Content Repurposing Engine

## Overview

Transforms blogs, notes, and articles into varied formats for different channels.

**Keywords**: content, repurposing, social media, threads, scripts, short-form, long-form

## Instructions

1. Accept original long-form content from the user
2. Identify the key ideas, insights, and value points
3. Ask the user for the target format (thread, script, summary, etc.) if not specified
4. Rewrite content adapted to the target platform's style and length
5. Maintain consistent tone and meaning throughout

## Output Format

- Platform-specific content blocks
- Structured sections with engaging headlines/hooks
- Clear separation between adapted pieces

## Constraints

- Preserve core meaning — do not distort the original message
- Avoid verbosity in short-form outputs
- Format must match the target channel's style conventions
