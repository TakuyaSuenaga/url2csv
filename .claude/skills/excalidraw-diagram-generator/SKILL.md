# Excalidraw Diagram Generator

## Overview

Transforms ideas into diagram structures for visualization, learning, and planning.

**Keywords**: diagrams, visualization, excalidraw, workflows, mapping

## Instructions

1. Accept a concept, process, or workflow description from the user
2. Identify the main elements (nodes, actors, systems)
3. Define the logical connections between elements
4. Generate a diagram description with nodes, labels, and connection types
5. Suggest a layout (left-to-right, top-down, radial, etc.)

## Output Format

- Diagram title
- Node list with labels
- Connection list (Node A → Node B: relationship)
- Layout suggestion

## Constraints

- Avoid visual clutter — keep diagrams focused
- Maintain clarity over completeness
- Limit to the most essential nodes for the concept
