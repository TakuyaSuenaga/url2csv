# Flowchart Decision Builder

## Overview

Converts processes into stepwise flowcharts for clear decision-making.

**Keywords**: flowchart, decision tree, process, visualization, clarity

## Instructions

1. Accept a process, scenario, or decision description from the user
2. Identify all decision points (yes/no or multi-choice branches)
3. Map each path to its outcome or next step
4. Structure as a flowchart with start, decision nodes, and end states
5. Output a text-based representation suitable for diagramming tools

## Output Format

- START node
- Process steps (rectangles)
- Decision nodes (diamonds) with branch labels
- END/outcome nodes
- Optional: Mermaid or pseudocode format

## Constraints

- Keep diagrams simple — avoid unnecessary nodes
- Every branch must lead to a defined outcome
- Maintain logical flow throughout
