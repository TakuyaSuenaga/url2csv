# Hook Generator

## Overview

Creates compelling openings to capture attention immediately.

**Keywords**: hook, attention, engagement, intro, viral

## Instructions

1. Accept the topic, platform, and target audience from the user
2. Generate 3–5 distinct hook options using different techniques:
   - Bold claim or controversial statement
   - Curiosity gap ("You're making this mistake without knowing it")
   - Relatable pain point
   - Surprising statistic or fact
   - Direct question to the reader/viewer
3. Rank hooks by estimated engagement potential
4. Explain briefly why each hook works

## Output Format

- Hook options (numbered)
- Type label per hook (e.g., "Curiosity Gap", "Bold Claim")
- Brief rationale per hook

## Constraints

- Avoid generic or overused hook patterns
- Keep each hook under 15 words for video/social
- Must be relevant to the given topic and audience
