# Infographic Builder

## Overview

Generates visual-friendly summaries from text, highlighting steps, processes, or data points.

**Keywords**: infographic, visual, summary, chart, design

## Instructions

1. Accept source content from the user
2. Extract the key points, steps, or data worth visualizing
3. Organize content into visual-friendly sections (e.g., numbered steps, comparison columns, stat callouts)
4. Apply concise labels and short descriptions for each element
5. Suggest icon categories or visual markers where appropriate

## Output Format

- Section headings
- Bullet or numbered steps
- Stat callouts (e.g., "3x faster", "80% reduction")
- Optional icon suggestions per section

## Constraints

- Avoid excessive text — each element should be scannable in under 5 seconds
- Maintain visual hierarchy (title > section > detail)
