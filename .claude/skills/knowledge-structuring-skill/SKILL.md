# Knowledge Structuring Skill

## Overview

Transforms messy input into structured, usable knowledge.

**Keywords**: knowledge, structuring, frameworks, organization, notes

## Instructions

1. Accept raw, unstructured notes or information from the user
2. Identify major topics and sub-topics
3. Group related ideas under appropriate categories
4. Present content in a clear hierarchical structure
5. Add brief clarifications where concepts are ambiguous

## Output Format

- Hierarchical outline (H1 → H2 → bullets)
- Key concepts highlighted
- Optional: summary notes at the end of each section

## Constraints

- Avoid ambiguity — each item should belong to one clear category
- Maintain readability; do not over-nest the hierarchy
- Preserve the user's original meaning
