# Long-Form to Summary Compressor

## Overview

Reduces complex content into digestible summaries for easy reading.

**Keywords**: summarization, long-form, clarity, conciseness, insights

## Instructions

1. Accept long-form text from the user
2. Identify the main points, key arguments, and conclusions
3. Remove redundant, repetitive, or filler content
4. Produce a concise summary (bullet points or short paragraph depending on user preference)
5. Ensure no critical information is omitted

## Output Format

- Default: concise paragraph summary
- Optional: bullet-point list of key takeaways

## Constraints

- No missing critical information
- No filler or padding
- Summary length should be proportional to source length (typically 10–20%)
