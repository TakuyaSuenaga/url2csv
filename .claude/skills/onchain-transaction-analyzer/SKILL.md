# Onchain Transaction Analyzer

## Overview

Decodes onchain data into human-readable explanations.

**Keywords**: blockchain, crypto, analysis, transactions, wallets

## Instructions

1. Accept a transaction hash, wallet address, or contract address from the user
2. Trace the wallet and token flows involved
3. Identify key actors (sender, receiver, contracts, protocols)
4. Explain each interaction in plain language
5. Summarize the overall purpose and outcome of the transaction

## Output Format

- Step-by-step explanation of the transaction
- Key actors and their roles
- Token movements (from → to, amount)
- Summary with plain-language interpretation

## Constraints

- Avoid jargon — explain technical terms when used
- Focus on clarity over technical completeness
- Highlight anything unusual or noteworthy
