# SCQA Writing Framework

## Overview

Transforms unstructured ideas into structured, high-engagement content. Perfect for educational material, storytelling, and technical explanations.

**Keywords**: writing, storytelling, SCQA, structured content, clarity, narrative, article, thread

## Core Framework

### Situation
- Establish context, current state
- Concise, clear, relevant

### Complication
- Introduce problem or tension
- Create curiosity

### Question
- Frame audience's key question
- Essential and natural

### Answer
- Deliver insight or solution
- Clear, actionable

## Instructions

1. Accept the user's raw input (idea, topic, or draft)
2. Break content into four SCQA sections
3. Keep each section concise and purposeful
4. Ensure smooth logical flow between sections
5. Avoid jargon; prioritize clarity

## Constraints

- No skipped sections
- No repetition across sections
- Conciseness prioritized over completeness
