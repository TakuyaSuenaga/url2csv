# Structured Copywriting Skill

## Overview

Produces persuasive, well-structured copy with strong hooks and calls to action.

**Keywords**: copywriting, marketing, social media, structured content, hooks, engagement

## Instructions

1. Identify the product, service, or idea to be promoted
2. Craft an attention-grabbing opening hook
3. Organize the main value points in a clear, logical flow
4. Include a strong, actionable call-to-action (CTA)
5. Review for readability and remove unnecessary filler

## Output Format

- Hook (1–2 sentences)
- Body sections with bullet highlights
- CTA at the end

## Constraints

- Maintain readability — avoid walls of text
- Do not overcomplicate the message
- Every sentence should serve a purpose
