# Tone & Style Enforcer

## Overview

Keeps all generated content in line with defined style guidelines or brand voice.

**Keywords**: style, tone, brand voice, consistency, clarity, writing

## Instructions

1. Ask the user to define their tone (e.g., professional, casual, authoritative, friendly) if not provided
2. Apply the defined tone uniformly across all content
3. Check for style inconsistencies (vocabulary, sentence length, formality level)
4. Adjust language, structure, and formatting to match the style guide
5. Output the revised content with a short note on changes made

## Constraints

- No deviation from the selected tone once defined
- Maintain readability at all times
- Do not change the meaning of the original content
