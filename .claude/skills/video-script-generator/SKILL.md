# Video Script Generator

## Overview

Produces structured scripts for short and long-form video content.

**Keywords**: video, scripts, hooks, engagement, pacing, content

## Instructions

1. Accept the topic, target audience, and desired video length from the user
2. Write an attention-grabbing opening hook (first 3–5 seconds)
3. Structure the main content into clear, paced sections
4. Include transition cues between sections
5. End with a strong call-to-action (subscribe, follow, visit, etc.)

## Output Format

- [HOOK] — opening line(s)
- [SECTION 1], [SECTION 2], ... — body content
- [CTA] — closing call-to-action
- Optional: on-screen text suggestions in brackets

## Constraints

- Avoid filler words and padding
- Keep sentences short for easy delivery
- Pacing should match the target platform (YouTube vs. TikTok vs. Reel)
